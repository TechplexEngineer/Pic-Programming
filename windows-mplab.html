Windows MPLAB Project Setup

  1. Download MPLab (v8.86) and install.

  2. Download HiTech C Compiler and install. (Choose lite mode and add to environment path)

  3. If MPLab was open during the install. Exit out of MPLab.

  4. Start MPLab from the icon on your desktop

  5. To tell MPLAP what PIC chip we have:

    1. In MPLAB IDE, click Configure -> Select Device.

    2. Choose a PIC16F887 from the Device dropdown menu

    3. click OK.

  6. To create a new project:

    1. by clicking Project -> New.... A dialog box will appear named New Project.

    2. In the Project Name text input box type 'lab 1'.

    3. In the Project Directory text input box create a new folder on your desktop and call it ece101

    4. Click OK. If the directory does not exist, click OK to create it.

  7. To tell MPLab how to compile our code:

    1. Now select the language toolsuite by clicking Project -> Select Language Toolsuite.

    2. In the Active Toolsuite dropdown box select HI-TECH Universal ToolSuite. Click on HI-TECH Compiler in the Toolsuite Contents text area.

    3. Click the Browse... button

    4. Navigate to c:/Program Files If you are on a 64bit computers you may need to look in c:/Program files (x86)

    5. Double click HI-TECH Software.

    6. Double click PICC.

    7. Double click 9.83.

    8. Double click bin. Double click picc.exe. Click OK.

  8. To get the starter files from the course website:.

    1. Go here to download files

    2. Right mouse click on each of the files and save them into your new directory.

  9. To add the starter files to our project:

    1. Show the project explorer by choosing: View -> Project.

    2. In the lab1.mcw project subwindow, right click lab1.mcp and select Add Files... Control click all library (.lpp) and header files (.h). then Open.

    3. In the lab1.mcw subwindow, right click library files and select Library Link Order... Click delay.lpp and click the down arrow twice to move delay.lpp to the bottom. Click OK.

    4. To enter code, open robot.c enter your code inside the main function;

  10. To setup the delay library: (The delay library needs to know how fast the PIC processor is.)

    1. Choose Project → build options → robot.c

    2. In the second field just left of the add button enter: _XTAL_FREQ=4000

    3. Click the add button.

    4. Close the build options window.

MPLAB Entering Code

  1. The first line of any program should be a comment with the following form: Your name, course, date, and lab day. For example:

    // Jane Doe ECE 101 01/25/2012 Tuesday

  2. Following comments are include header files. Always include pic.h. (#include <pic.h>)

  3. If you are not using robot.h which already includes the config bits, you'll need to add the following line in the file where void main(void) is defined, immediately after all include & preprocessor directives.

    __CONFIG(FOSC_INTRC_NOCLKOUT & WDTE_OFF & PWRTE_OFF & MCLRE_OFF & CP_OFF & CPD_OFF & BOREN_OFF & IESO_OFF & FCMEN_OFF & LVP_OFF & DEBUG_OFF);

MPLAB Compiling

  1. Click Project -> Build All. Check the Output subwindow, Build tab for error and/or warnings. Fix any and all errors and warnings. Advisory statements may be safely ignored.

MPLAB Programming

  1. Sign out a PICKIT2 programmer from a TA.

  2. Connect power to you breadboard after a TA has checked the wiring.

  3. Connect both ends of the USB cable.

  4. Connect your programmer to the 6 pin header on your breadboard. Be sure to orient the programmer correctly.

  5. Click Programmer -> Select Programmer -> PicKit2.

  6. To program your chip click Programmer -> Program. All verifies should pass.
